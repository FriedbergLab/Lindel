Introduction

Lindel is a logistic regression model that predicts all the possible indels (<30 bp) that result from Cas9 induced NHEJ. It takes an input of 65 bp sequence with the cleavage site located at the 30 bp of the sequence and predicts ~420 unique deletions and all possible 1-2 bp insertions..

Using Lindel

Webtool

We developed a webtool to show the distribution of most frequent CRISPOR.

Integrated in CRISPOR

Lindel has been integrated into guide design tool CRISPOR.

Installing local Lindel

Lindel has been wrapped as a python package. It includes the functions to parse input sequence and models to predict the frequencies of all indels. Python 3.5 or higher is required to run Lindel. Version for Python 2.7 will come out soon.
